# 📊 Event Sales Analysis Project (Excel)

## 🔗 Project Overview
This project presents a comprehensive analysis of event ticket sales data using Microsoft Excel. It highlights strong analytical, problem-solving, and data visualization skills, making it highly relevant for roles in business analytics and data analysis.

---

## 🧠 Key Features
- Cleaned and structured raw event data for analysis
- Created dynamic fields using advanced Excel formulas
- Built interactive Pivot Tables for multi-dimensional analysis
- Calculated key performance indicators (KPIs): Revenue, Margin (%), Sales Contribution (%)
- Applied conditional formatting to highlight trends and anomalies
- Enabled interactive filtering by city, genre, and category

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Excel Functions: SUM, AVERAGE, IF, COUNT, COUNTA, etc.

---

## 📈 Key Insights
- Identified top-performing cities and event genres driving revenue
- Analyzed monthly sales trends to detect peak performance periods
- Highlighted high-demand events using visual cues and formatting
- Evaluated profitability through margin analysis
